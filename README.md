## 说明
此项目主要用来学习，编辑器，图片上传，和webpack的基本配置！

## 技术栈
```
主要列举
react@16.4.1
react-router-dom@4.3.1
axios@0.18.0
mongoose@5.1.3
webpack@4.8.3
react-draft-wysiwyg@1.12.13
```

## 项目运行

```
git clone https://github.com/tianjie23/webManage.git

 cd react-webManage

 npm i  或者运行  yarn(推荐)
 
 npm server
 
 npm start

 npm run build （发布）
```
